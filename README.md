# Equity-Valuation-Investment-Recommendation-

## 📌 Overview
This project presents a comprehensive equity research and investment analysis of selected Indian IT sector companies using both fundamental and technical approaches. The study applies the EIC (Economy–Industry–Company) framework, valuation models, and technical indicators to derive actionable investment insights.

---

## 🎯 Objective
- Evaluate the investment potential of selected IT companies  
- Estimate intrinsic value using multiple valuation techniques  
- Analyze market trends and price behavior  
- Provide BUY / HOLD / SELL recommendations based on data-driven insights  

---

## 🏢 Companies Analyzed
- Tata Elxsi Ltd.  
- Oracle Financial Services Software Ltd. (OFSS)  
- HCL Technologies Ltd.  

---

## ⚙️ Methodology

### 1️⃣ Fundamental Analysis
- EIC Framework (Economy–Industry–Company)  
- Financial performance evaluation  
- Industry and macroeconomic analysis  

### 2️⃣ Valuation Models
- Discounted Cash Flow (DCF)  
- Relative P/E Valuation  
- Benjamin Graham Intrinsic Value Model  

### 3️⃣ Technical Analysis
- Moving Averages (MA20 & MA50)  
- Trend Analysis (Bullish / Bearish / Consolidation)  

---

## 📈 Key Insights
- HCL Technologies identified as undervalued with strong upside potential  
- Tata Elxsi & OFSS found to be overvalued based on intrinsic valuation  
- Technical analysis indicates:  
  - Tata Elxsi → Bearish trend  
  - OFSS → Consolidation phase  
  - HCL Tech → Emerging bullish momentum  

---

## 🏆 Final Recommendation

| Company | Recommendation | Rationale |
|--------|--------------|----------|
| HCL Technologies | BUY | Undervalued + Strong technical momentum |
| Tata Elxsi | HOLD | Strong fundamentals but overvalued |
| OFSS | HOLD | Stable performance but limited upside |
